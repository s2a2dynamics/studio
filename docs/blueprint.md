# **App Name**: Twilio Firebase Notifier

## Core Features:

- HTTP Trigger: A public web link that triggers the Cloud Function when visited.
- Cloud Function Execution: Executes Node.js code within a Firebase Cloud Function upon HTTP request.
- Twilio API Call: Invokes the Twilio API to send a WhatsApp message.
- Secret Management: Securely stores Twilio credentials (Account SID and Auth Token) in Firebase environment configuration.
- WhatsApp Message Sending: Sends a WhatsApp message via Twilio's API.

## Style Guidelines:

- Primary color: Soft blue (#A0D2EB) to reflect reliability and communication.
- Background color: Light gray (#F0F4F7) to ensure readability and a clean interface.
- Accent color: Teal (#008080) for key actions and visual cues.
- Font: 'Inter', a sans-serif font that will be used for the headlines and body text to create a modern and clean aesthetic.
- Use simple, outline-style icons related to notifications and messaging.
- Clean and straightforward layout with a focus on clarity.
- Subtle animations on button hover states to enhance interactivity.
